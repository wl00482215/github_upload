在終端機打下列指令
1.在目前資料夾進行初始化
git init 

2.將github上的資料夾下載進本機
git clone https://github.com/wl00482215/github_upload.git

3.在github下載的資料夾裡建立使用者資料
git config user.email "wl00482215@gmail.com"
git config user.name "willy"

4.將檔案test.txt的內容，加入資料庫的索引
git add test.txt

5.將目前資料庫中建立一個版本名稱
git commit -m "test_commit_1"

6.取消暫存區 git add添加的項目
git reset test.txt

7.看git的log檔，看完要記得按q離開
git log
